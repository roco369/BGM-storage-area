# BGM-storage-area

自作ゲームや各種プロジェクトで使用するBGM・音源データを管理・公開しているリポジトリです。
GitHub Pagesを利用して、各音源ファイルへ直接アクセスできるようにしています。

## 🌐 公開ページ
以下のURLから、ホストされている音源を確認できます。
[https://roco369.github.io/BGM-storage-area/](https://roco369.github.io/BGM-storage-area/)

---

## 📂 収録内容
* **Audio Files**: ゲーム実装用のmp3等の音源データ
* **Index**: ブラウザ上で音源を確認するためのページ構成
